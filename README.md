# 概要/Overview
- 二桁以内の数字を2つ入力し、ボタンをクリックすると、入力された値の倍数が小さい順に表示
- 各々倍数と一緒にfizz、buzzと表示
- 両方の倍数である時にはfizzBuzzと表示

# 制作背景
JavaScriptの条件分岐、繰り返し処理、ループ処理、DOMで学んだことをFizzBuzzプログラムでアウトプットしています。
